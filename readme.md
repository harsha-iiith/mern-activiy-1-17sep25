# Mern Lab 1

## clone the repo

```bash
git clone https://github.com/AadityaNarain2003/Mern_Lab_1_
```

## install dependencies

```bash
# Rename folder to your rollnumber
cd your-rollnumber/backend
npm install
npm install nodemon mongoose
```

## run the server

```bash
npm run dev
```

## run the tests

```bash
npm run test
```
